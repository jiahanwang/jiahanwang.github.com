Introduction

This is a BS system with a rich client interface and a powerful website background system. Student users write their compositions online; then teachers mark the compositions by adding tags. Teachers can also assign a task of certain topic for certain class.

Feathers

Struts

The back end of this system is based on Struts Frameworks. And JDBC is applied to connect the MySQL database.

jQuery UI

To enhance the usability, several jQuery UI widgets are used. Accordian can help users to compare the origin and the revised compositions. Tabs could show multi pages in a iframe.

JFreeChart

The system stores the compositions by a XML tool. Then JFreeChart plugin will be applied to analyze errors and generate reports to teachers.
