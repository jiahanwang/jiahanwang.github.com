Introduction

This web application evaluates a submitted Japanese composition from vocabulary, grammar and content.

Feathers

Composition Evaluation

Call the Yahoo! APIs by PHP script to evaluate every submitted composition. Get the XML result, parse the data and send to front end by Ajax.

All in One Page

The whole application is just one web page. All the data transmission is done by Ajax and the DOM tree is manipulated by jQuery.
