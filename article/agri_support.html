Introduction

This system was developed by adopting the thinking of software engineering. We successfully developed all processes from selling agricultural products to buying agricultural products. Its main functions are buying agricultural production, managing agricultural production materials and so on. There are two different login statuses with different login rights: one is seller and the other is producer, they can do what they want within their power.

Feathers

SSH

This system is based on classic SSH frameworks of Java. Struts2 is applied to separate the model from the view and the controller. The Inversion of Control container in Spring is chosen to manage lifecycle of Java objects for us. Hibernate is also integrated to map an object-oriented domain model to a traditional relational database.

Friendly Interface

Several UI components are applied to enhance user experience, such as accordion from jQuery to show different operations and TableSorter plugin to sort the table by each headers.
