Back End Skills

Java

Java was not born just for web; however, it turned out to be one of the most popular languages to build back-end for web applications. The most exciting thing about Java is its spirit of open source. Since Java was created, hundreds of programmers in the world have developed enormous free jar packages. As the result, Java is enhanced and extended. The abundant frameworks might be another advantage of Java. SSH is common-used in web development. It is a combination of three frameworks aiming at different layers of a website.

  • Struts: The goal of Struts is to separate the model from the view and the controller. It is very easy to manage these three parts by maintaining a config.xml file. Furthermore, its powerful custom tag library allows Struts to read and write the content of JavaBeans from the presentation layer without the need for any embedded Java code.
  • Spring: Spring might be the most elegant and innovative framework ever. The most important feather is the Inversion of Control container done mainly via dependency injection. You just have to register a class in the configuration file and, Spring will manage lifecycle of Java objects. Spring has its own AOP framework which is very helpful for transaction management, security, and remote access.
  • Hibernate: Hibernate is an object-relational mapping (ORM) library, providing a framework for mapping an object-oriented domain model to a traditional relational database. Its primary feature is mapping from Java classes to database tables. To manage mapping is easy, also by maintaining a XML configuration file.

PHP

Just as its P-family competitors - ASP and JSP, PHP is a server-side scripting language and it can be embedded into an HTML source document rather than calling an external file to process data. However, PHP is light-weight and easier to develop. To connect to a database especially MySQL is easygoing. Also the superglobals are precise tools for us. $_REQUEST, $_POST, $_GET let us to fetch data in http request conveniently. Client authentication is no longer complex with $_COOKIE and $_SESSION.

C/C++

For a front end developer, C and C++ usually aren’t used. Nonetheless, these two programming languages are the base of other top-level languages. Many engines or interpreters are built in C or C++. These two are also the first choice for beginners. I stepping into the world of Computer Science guided by C language and, C++ showed me the door to OOP programming.
