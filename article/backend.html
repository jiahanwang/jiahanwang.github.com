Java

Java was not born just for web; however, it turned out to be one of the most popular languages to build back-end for web applications. Well thanks to Apache Foundation, Java is actually popular in almost very field. The abundant open source packages and frameworks are a huge advantage of Java. SSH is common-used in web development, which is a combination of three frameworks aiming at different layers of a website.

  • Struts: The goal of Struts is to separate the model from the view and the controller(MVC of course). It is very easy to manage these three parts by maintaining a config.xml file. Furthermore, its powerful custom tag library allows Struts to read and write the content of JavaBeans from the presentation layer without the need for any embedded Java code.
  • Spring: Spring might be the most elegant and innovative framework ever. The most important feather is the Inversion of Control container done mainly via dependency injection. Spring also supports AOP, which is very helpful for transaction management, security, and remote access.
  • Hibernate: Hibernate is an object-relational mapping (ORM) library, providing a framework for mapping an object-oriented domain model to a traditional relational database. Its primary feature is mapping from Java classes to database tables. To manage mapping is easy, also by maintaining a XML configuration file.

PHP

Just as its P-family competitors - ASP and JSP, PHP is a server-side scripting language. PHP is light-weight and super easy to use. PHP and MySQL are good buddies; connection to MySQL is supported by built-in APIs. Also the superglobals are fantastic tools for us. $_REQUEST, $_POST, $_GET let us to fetch data in http request conveniently. Client authentication is no longer complex with $_COOKIE and $_SESSION. Another feature of PHP is it is always happy to copy other programming languages. But, the most important feature is PHP has one of the best online API documents in the world. (Yes. This is very important.)

MySQL

Easy-to-use open source relational database. Good companion for PHP (where the term LAMP comes from). I've used MySQL for a couple of projects. phpMyAdmin is often used to handle the administration; however, Oracle now offers a default GUI which is pretty nice.
