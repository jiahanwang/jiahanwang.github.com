Feathers

jQuery Plugin – Scrollable

The main function is implemented by a robust jQuery plugin – Scrollable. The basic principle of this plugin is that the height of the inner <div class=”itmes”> is very large while the overflow attribute is hidden. Every time to show next <div class=”itme”> in <div class=”itmes”>, the top position will be added by one height of the item.

Mimic Ajax

The loading of images is asynchronous; however it isn’t implemented by the XMLHttpRequest object. We can create a <img> without the src attribute, hide it and let the loading begin by setting the src attribute; at last show the image when the loading is finished.
