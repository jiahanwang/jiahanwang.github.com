Introduction

This web application aims to assist Baidu editors to edit the popular words searched by users, as the display area of mobile devices is limited.

Feathers

Responsive CSS & JS

This web application needs to adapt to PC and mobile devices, thus responsive style sheets are applied. Moreover, as user behavior of tablets, phones with touch screen differs greatly from that of traditional PCs, JavaScript is also written in a responsive way.

UUAP

UUAP(User Unify Authentication Platform)is a user data management and authentication system developed by Baidu. It has two sub-systems: UIC（User Information Center）and SSO（Single Sign On）. By applying php CAS, a web system could use these services without worrying user data management and safety issues.
